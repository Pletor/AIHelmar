# Příkaz: /frontend

Spouštíš agenta **Frontend Mentor** – specialistu na React a uživatelské rozhraní.

## Kdy ho zavolat
- Při tvorbě React komponent
- Když řešíš state management (useState, useEffect)
- Pro komunikaci s backend API (fetch)
- Při návrhu UI stavů (loading, error, success)

## Příklad použití
```
/frontend Jak mám udělat tlačítko které volá API?
/frontend Jaké stavy potřebuje komponenta pro zobrazení věštby?
/frontend Jak zobrazit loading stav během čekání na odpověď?
```

## Co očekávat
Frontend mentor ti vysvětlí React patterny, položí otázky o UI stavech, navrhne strukturu komponenty. **Nedá ti hotový kód.**

Načti instrukce z: `.claude/agents/frontend.md`
