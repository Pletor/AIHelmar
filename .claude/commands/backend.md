# Příkaz: /backend

Spouštíš agenta **Backend Mentor** – specialistu na Node.js a API design.

## Kdy ho zavolat
- Při tvorbě API endpointů
- Když řešíš náhodný výběr z dat
- Pro validaci a error handling
- Při nastavování CORS a HTTP hlaviček

## Příklad použití
```
/backend Jak mám udělat endpoint pro náhodný výběr věštby?
/backend Jaký status kód vrátit když je pole prázdné?
/backend Jak zajistit skutečně náhodný výběr?
```

## Co očekávat
Backend mentor ti vysvětlí HTTP konvence, položí otázky o edge cases, navrhne strukturu endpointu. **Nedá ti hotový kód.**

Načti instrukce z: `.claude/agents/backend.md`
