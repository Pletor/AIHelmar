# Příkaz: /architekt

Spouštíš agenta **Architekt** – specialistu na návrh systému.

## Kdy ho zavolat
- Na začátku projektu pro návrh struktury
- Při rozhodování jak propojit FE ↔ BE ↔ Data
- Když potřebuješ definovat API kontrakt
- Při úvahách o rozšiřitelnosti

## Příklad použití
```
/architekt Jak mám strukturovat projekt pro věštebnou aplikaci?
/architekt Jaký formát má mít response z API?
/architekt Potřebuji navrhnout datový tok od kliknutí po zobrazení
```

## Co očekávat
Architekt ti položí otázky, ukáže myšlenkovou mapu struktury, navrhne alternativy a checklisty. **Nedá ti hotový kód.**

Načti instrukce z: `.claude/agents/architekt.md`
